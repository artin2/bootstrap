Exercise 2: Bootstrap's "Layout Grid"

Bootstrap's grid system is very simple to use. Let's try messing around with it:

  • Give the next div a row class. (Containers can have 1 or more rows.)

Exercise 3: Forms

Bootstrap helps with forms too. However, forms follow some rules, so pay attention to that in this exercise:

  • Every form element should have an id. Add id="formControl1" to the text field below and id="formControl2" to the select box below.
  • The label for the form control (My Textbox and My Select) should use the label element. Change the div to a label for both form controls.
  • Every label should have a for attribute equal to the id of its corresponding form control. Add for="formControl1" to the label for the text field and for="formControl2" to the label for the select box.
  • To activate the Bootstrap styling, add the form-group class on the div surrounding each form control, and add form-control to the text field and the select.
  • Now add a new form from scratch.
